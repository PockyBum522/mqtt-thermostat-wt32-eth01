# mqtt-thermostat-wt32-eth01
Making my own thermostat from scratch with a ESP32


# Wiring:

## SHT30:

SHT30  | Wire Color | ESP8266 Pin | ESP32 Pin
VIN    | Red        | 3.3v        | 3.3v
SCL    | Yellow     | D1          | IO 22
SDA    | White      | D2          | IO 21
Ground | Green      | Ground      | Ground
